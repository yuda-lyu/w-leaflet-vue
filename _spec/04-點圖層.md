# 04 - 點圖層 (pointSets)

## 1. 概述

點圖層支援兩種顯示模式：**圓點 (circle)** 與 **圖標 (icon)**，可在集合層級或個別點層級設定樣式。

---

## 2. 資料結構

### 點集合 (pointSet)

| 屬性 | 型別 | 預設值 | 說明 |
|------|------|--------|------|
| `title` | `String` | `''` | 集合標題 |
| `msg` | `String` | `''` | 集合說明 |
| `order` | `Number` | `null` | 排序用（圖層顯隱面板排列順序） |
| `type` | `String` | `'circle'` | 顯示類型 `'circle'` 或 `'icon'` |
| `size` | `Number` | `10` | 圓點半徑 (px) |
| `lineColor` | `String` | `'rgba(255,255,255,1)'` | 框線顏色 |
| `lineWidth` | `Number` | `1` | 框線寬度 |
| `fillColor` | `String` | `'rgba(0,150,255,0.65)'` | 填充顏色 |
| `iconSrc` | `String` | Google Map 預設點圖標 | 圖標來源 (base64/URL) |
| `iconSize` | `Array` | `[24, 40]` | 圖標尺寸 `[寬, 高]` (px) |
| `iconAnchor` | `Array` | `[iconSize[0]/2, iconSize[1]]` | 圖標定位點 |
| `popupAnchor` | `Array` | `[0, -iconSize[1]/1.5]` | Popup 指向偏移 |
| `visible` | `Boolean` | `true` | 是否顯示 |
| `points` | `Array` | `[]` | 各點資料陣列 |

### 單點 (point)

| 屬性 | 型別 | 預設值 | 說明 |
|------|------|--------|------|
| `title` | `String` | `''` | 點標題 |
| `msg` | `String` | `''` | 點說明 |
| `latLng` | `Array` | `[]` | 座標 `[lat, lng]` |
| `type` | `String` | 繼承 pointSet | 顯示類型（可覆蓋集合設定） |
| `size` | `Number` | 繼承 pointSet | 圓點半徑 |
| `lineColor` | `String` | 繼承 pointSet | 框線顏色 |
| `lineWidth` | `Number` | 繼承 pointSet | 框線寬度 |
| `fillColor` | `String` | 繼承 pointSet | 填充顏色 |
| `iconSrc` | `String` | 繼承 pointSet | 圖標來源 |
| `iconSize` | `Array` | 繼承 pointSet | 圖標尺寸 |
| `iconAnchor` | `Array` | 繼承 pointSet | 圖標定位點 |
| `popupAnchor` | `Array` | 繼承 pointSet | Popup 偏移 |

### 簡化格式

點可以直接用陣列表示：`[lat, lng]`，會自動轉換為物件 `{ latLng: [lat, lng] }`

---

## 3. 樣式繼承機制

```
個別點屬性 → 若無則繼承 pointSet 屬性 → 若無最終使用預設值
```

- **type**: 個別點的 type > pointSet 的 type > 'circle'
- **size/radius**: 個別點的 size > pointSet 的 size > 10
- **lineColor**: 個別點的 lineColor > pointSet 的 lineColor > 'rgba(255,255,255,1)'
- **lineWidth**: 個別點的 lineWidth > pointSet 的 lineWidth > 1
- **fillColor**: 個別點的 fillColor > pointSet 的 fillColor > 'rgba(0,150,255,0.65)'
- **icon**: 個別點有完整 icon 參數則優先使用，否則用 pointSet 的 icon

---

## 4. 渲染邏輯

### 圓點模式 (type==='circle')

使用 `l-circle-marker`：

```js
style = {
    radius,        // 圓點半徑
    fillColor,     // 填充色
    fillOpacity: 1,// 填充透明度（強制為 1）
    color: lineColor, // 框線色
    weight: lineWidth  // 框線寬
}
```

### 圖標模式 (type==='icon')

使用 `l-marker` + Leaflet `Icon`：

```js
new Icon({
    iconUrl: iconSrc,
    iconSize,
    iconAnchor,
    popupAnchor,
    tooltipAnchor
})
```

## 5. 事件支援

### 點擊事件

| 屬性 | 路徑 | 說明 |
|------|------|------|
| `pointSetsClick` | `opt.pointSetsClick` | 全域點擊回呼函數 |

回呼參數：

```js
{
    ev,         // 原始事件
    point,      // 當前點資料
    kpoint,     // 點索引
    pointSet,   // 所屬集合
    kpointSet,  // 集合索引
    pointSets   // 所有集合
}
```

| 插槽名稱 | 作用域 | 說明 |
|----------|--------|------|

---

## 7. z-index 處理

- `l-marker` 支援 `zIndexOffset` 屬性
- 未指定時 marker 重繪可能改變 z-index
- `l-circle-marker` 不支援 `zIndexOffset`

---

## 8. 資料變更偵測

1. 使用 `debounce` 防抖
2. 透過 `omit(v, 'visible')` 排除 visible 屬性後比對
3. 僅在資料實質變更時重新處理（`isEqual` 比對 `effPointSetsTemp`）
