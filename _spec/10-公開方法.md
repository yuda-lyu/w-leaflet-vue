# 10 - 公開方法 (Public Methods)

## 概述

以下方法可透過元件 `ref` 從外部呼叫。

---

## 1. getMapObject()

取得底層 Leaflet Map 物件。

```js
let mapObject = this.$refs.leafletMap.getMapObject()
```

**回傳值**: Leaflet Map 物件

---

## 2. panTo(latLng, opt)

平移地圖至指定座標，支援偏移比率與自訂計算。

### 參數

| 參數 | 型別 | 說明 |
|------|------|------|
| `latLng` | `Array` | 目標座標 `[lat, lng]` |
| `opt` | `Object` | 選用設定 |
| `opt.ratioHorizontal` | `Number` | 水平縮小比率（左側起算） |
| `opt.ratioVertical` | `Number` | 垂直縮小比率（上方起算） |
| `opt.funLatLng` | `Function` | 自訂座標計算函數 |

### 偏移計算邏輯

```
1. 取得當前地圖範圍 (getBounds)
2. 計算經度/緯度範圍
3. 反算以 latLng 為中心的角點
4. 若有 ratioHorizontal：水平方向縮小可視範圍
5. 若有 ratioVertical：垂直方向縮小可視範圍
6. 計算新中心點
7. 若有 funLatLng：使用自訂函數微調
8. 呼叫 mapObject.panTo(latLngNew)
```

### funLatLng 回呼參數

```js
funLatLng(latLng, {
    bds, lngRange, latRange,
    lngMin, lngMax, latMin, latMax,
    ratioHorizontal, ratioVertical,
    latLngNew
})
```

---

## 3. centerTo(latLng)

`panTo` 的簡化版本，直接將地圖中心移至指定座標。

```js
this.$refs.leafletMap.centerTo([25.0330, 121.5654])
```

---

## 4. closePopup()

關閉當前開啟的 Popup。

```js
this.$refs.leafletMap.closePopup()
```

---

## 5. popupPoint(params)

程式化開啟指定點的 Popup。

```js
let latLng = this.$refs.leafletMap.popupPoint({ point: pointData })
```

**回傳值**: `[lat, lng]` 或 `null`

---

## 6. popupPolyline(params)

程式化開啟指定折線的 Popup。

```js
let latLng = this.$refs.leafletMap.popupPolyline({ polylineSet: polylineData })
```

---

## 7. popupPolygon(params)

程式化開啟指定多邊形的 Popup。

```js
let latLng = this.$refs.leafletMap.popupPolygon({ polygonSet: polygonData })
```

---

## 8. popupFeatureById(id)

依據內部產生的 ID 開啟指定 Feature 的 Popup。

```js
let latLng = this.$refs.leafletMap.popupFeatureById('pointSet-0-point-2')
```

**ID 格式**:
- 點: `pointSet-{kpointSet}-point-{kpoint}`
- 折線: `polylineSet-{kpolylineSet}`
- 多邊形: `polygonSet-{kpolygonSet}`
- GeoJSON: `geojsonSet-{kgeojsonSet}`
- 等值線: `contourSet-{kcontourSet}`

---

## 9. modifyItemsVisible(fun)

批次修改圖層可見性。

```js
this.$refs.leafletMap.modifyItemsVisible((item, index) => {
    return item.name === '測站'  // 回傳 true/false 控制 visible
})
```

### 參數

| 參數 | 型別 | 說明 |
|------|------|------|
| `fun` | `Function` | 接收 `(item, index)` 回傳 `Boolean` |

### 行為

1. 遍歷所有 items
2. 呼叫 `fun(item, k)` 取得新 visible 值
3. 使用 `set()` 同步內部與外部 `opt` 的 visible
