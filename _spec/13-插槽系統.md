# 13 - 插槽系統 (Slots)

## 1. 概述

組件透過 Vue 2 的 scoped slots 機制，讓外部可以自訂各圖層的 Tooltip 與 Popup 內容。

---

## 2. 插槽清單

### 點圖層

| 插槽名稱 | 作用域參數 | 說明 |
|----------|-----------|------|
| `point-tooltip` | `{ point, pointSet }` | 自訂點的 Tooltip 內容 |
| `point-popup` | `{ point, pointSet }` | 自訂點的 Popup 內容 |

### 折線圖層

| 插槽名稱 | 作用域參數 | 說明 |
|----------|-----------|------|
| `polyline-tooltip` | `{ polylineSet, polylineSets }` | 自訂折線的 Tooltip 內容 |
| `polyline-popup` | `{ polylineSet, polylineSets }` | 自訂折線的 Popup 內容 |

### 多邊形圖層

| 插槽名稱 | 作用域參數 | 說明 |
|----------|-----------|------|
| `polygon-tooltip` | `{ polygonSet, polygonSets }` | 自訂多邊形的 Tooltip 內容 |
| `polygon-popup` | `{ polygonSet, polygonSets }` | 自訂多邊形的 Popup 內容 |

### GeoJSON 圖層

| 插槽名稱 | 作用域參數 | 說明 |
|----------|-----------|------|
| `geojson-tooltip` | `{ geojsonSet, geojsonSets }` | 自訂 GeoJSON 的 Tooltip 內容 |
| `geojson-popup` | `{ geojsonSet, geojsonSets }` | 自訂 GeoJSON 的 Popup 內容 |

### 等值線圖層

| 插槽名稱 | 作用域參數 | 說明 |
|----------|-----------|------|
| `contour-tooltip` | `{ contourSet, contourSets }` | 自訂等值線的 Tooltip 內容 |
| `contour-popup` | `{ contourSet, contourSets }` | 自訂等值線的 Popup 內容 |

---

## 3. 使用範例

```html
<WLeafletVue :opt="mapOpt">

    <!-- 自訂點的 Popup -->
    <template v-slot:point-popup="{ point, pointSet }">
        <div style="padding: 10px;">
            <h3>{{ point.title }}</h3>
            <p>{{ point.msg }}</p>
            <p>座標: {{ point.latLng }}</p>
        </div>
    </template>

    <!-- 自訂 GeoJSON 的 Tooltip -->
    <template v-slot:geojson-tooltip="{ geojsonSet }">
        <div>{{ geojsonSet.title }}</div>
    </template>

</WLeafletVue>
```

## 4. 實作細節

### 點圖層插槽傳遞路徑

```
WLeafletVue
  └── l-marker / l-circle-marker
       ├── TooltipPoint (子組件)
       │    └── slot → 轉發至 point-tooltip
       └── PopupPoint (子組件)
            └── slot → 轉發至 point-popup
```

### 區域圖層插槽傳遞路徑

```
WLeafletVue
  └── l-polygon / l-polyline
       ├── TooltipRegion (子組件)
       │    └── slot → 轉發至 xxx-tooltip
       └── PopupRegion (子組件)
            └── slot → 轉發至 xxx-popup
```

### GeoJSON / 等值線插槽傳遞路徑

```
WLeafletVue
  └── LwGeoJson / LwContour (子組件)
       ├── v-slot:tooltip → 轉發至 xxx-tooltip
       └── v-slot:popup → 轉發至 xxx-popup
```

---

## 5. 注意事項

- 所有插槽均為 scoped slots（作用域插槽）
- 插槽內容預設為空，不提供則不顯示 Tooltip/Popup
- Tooltip 與 Popup 的顯示方式由各子組件（PopupPoint, TooltipPoint 等）控制
- CSS 已透過 `::v-deep` 修改 Leaflet 預設的 popup/tooltip 樣式
