# 09 - 點陣圖圖層 (imageSets)

## 1. 資料結構

### 影像集合 (imageSet)

| 屬性 | 型別 | 預設值 | 說明 |
|------|------|--------|------|
| `title` | `String` | `''` | 集合標題 |
| `msg` | `String` | `''` | 集合說明 |
| `order` | `Number` | `null` | 排序用數字 |
| `visible` | `Boolean` | `false` | 是否顯示 |
| `image` | `Object` | (必填) | 影像資料物件 |

### image 物件

| 屬性 | 型別 | 說明 |
|------|------|------|
| `url` | `String` | 影像 URL（必填） |
| `lngMin` | `Number` | 影像左邊界經度（必填） |
| `lngMax` | `Number` | 影像右邊界經度（必填） |
| `latMin` | `Number` | 影像下邊界緯度（必填） |
| `latMax` | `Number` | 影像上邊界緯度（必填） |

---

## 2. 邊界計算

```js
bounds = [
    [latMin, lngMin],   // 西南角
    [latMax, lngMax]    // 東北角
]
```

---

## 3. 渲染

使用 `l-image-overlay`：

```html
<l-image-overlay
    :url="imageSet.image.url"
    :bounds="imageSet.bounds"
/>
```

---

## 4. 資料驗證

- `image` 必須為有效物件 (iseobj)
- `url` 必須為有效字串 (isestr)
- `lngMin`, `lngMax`, `latMin`, `latMax` 必須均為數字 (isnum)
- 驗證失敗時拋出 Error

---

## 5. 事件

| 屬性 | 路徑 | 說明 |
|------|------|------|
| `imageSetsClick` | `opt.imageSetsClick` | 全域影像點擊回呼 |
