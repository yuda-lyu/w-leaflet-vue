# 05 - 折線圖層 (polylineSets)

## 1. 資料結構

### 折線集合 (polylineSet)

| 屬性 | 型別 | 預設值 | 說明 |
|------|------|--------|------|
| `title` | `String` | `''` | 集合標題 |
| `msg` | `String` | `''` | 集合說明 |
| `order` | `Number` | `null` | 排序用數字 |
| `lineColor` | `String` | `'rgba(0,150,255,1)'` | 框線顏色 |
| `lineColorHover` | `String` | 同 lineColor | 滑鼠移入框線顏色 |
| `lineWidth` | `Number` | `3` | 框線寬度 |
| `lineWidthHover` | `Number` | 同 lineWidth | 滑鼠移入框線寬度 |
| `latLngs` | `Array` | `[]` | 折線座標（支援 polyline/multiPolyline） |
| `visible` | `Boolean` | `true` | 是否顯示 |

---

## 2. 樣式物件

### 一般樣式

```js
{
    color: lineColor,    // 線條顏色
    weight: lineWidth    // 線條寬度
}
```

### Hover 樣式

```js
{
    color: lineColorHover,
    weight: lineWidthHover
}
```

---

## 3. 渲染

使用 `l-polyline`：

```html
<l-polyline
    :ref="polylineSet.id"
    :latLngs="polylineSet.latLngs"
    v-bind="polylineSet.style"
    @click="clickPolyline"
/>
```

---

## 4. 事件

| 屬性 | 路徑 | 說明 |
|------|------|------|
| `polylineSetsClick` | `opt.polylineSetsClick` | 全域折線點擊回呼 |

回呼參數：

```js
{
    ev,              // 原始事件
    polylineSet,     // 當前折線集合
    kpolylineSet,    // 集合索引
    polylineSets     // 所有集合
}
```

---

## 5. 插槽

| 插槽名稱 | 作用域 | 說明 |
|----------|--------|------|
| `polyline-tooltip` | `{ polylineSet, polylineSets }` | 自訂 Tooltip |
| `polyline-popup` | `{ polylineSet, polylineSets }` | 自訂 Popup |

使用 `TooltipRegion` 與 `PopupRegion` 子組件。
