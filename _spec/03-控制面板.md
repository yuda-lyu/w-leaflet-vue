# 03 - 控制面板

## 1. 概述

組件提供 6 種控制面板，每個面板皆可獨立設定顯示/隱藏、位置、尺寸。

---

## 2. 羅盤面板 (panelCompassRose)

| 屬性 | 路徑 | 型別 | 預設值 | 說明 |
|------|------|------|--------|------|
| show | `opt.panelCompassRose.show` | `Boolean` | `false` | 是否顯示羅盤 |
| position | `opt.panelCompassRose.position` | `String` | `'topright'` | 位置 |
| size | `opt.panelCompassRose.size` | `Number` | `120` | 羅盤尺寸 (px) |
| withPanel | `opt.panelCompassRose.withPanel` | `Boolean` | `false` | 是否帶底板 |
| iconSrcLight | `opt.panelCompassRose.iconSrcLight` | `String` | `uiRes.iconCompassRoseLight` | 淺色圖標 |
| iconSrcDark | `opt.panelCompassRose.iconSrcDark` | `String` | `uiRes.iconCompassRoseDark` | 深色圖標 |
| iconSrc | `opt.panelCompassRose.iconSrc` | `String` | `null` | 自訂圖標（優先） |

### 圖標選取邏輯 (computed: useIconCompassRose)

```
1. 若 iconSrc 有值 → 直接使用
2. 若 withPanel=true → 使用 iconSrcDark
3. 否則依底圖 colorShade：
   - 'dark' → 使用 iconSrcLight
   - '' 或 'light' → 使用 iconSrcDark
```

---

## 3. 縮放按鈕面板 (panelZoom)

| 屬性 | 路徑 | 型別 | 預設值 | 說明 |
|------|------|------|--------|------|
| show | `opt.panelZoom.show` | `Boolean` | `true` | 是否顯示 |
| position | `opt.panelZoom.position` | `String` | `'bottomleft'` | 位置 |

- 使用 Leaflet 原生 `l-control-zoom`
- 必須停用 LMap 內建 `zoomControl: false`，改由此面板控制位置

---

## 4. 比例尺面板 (panelScale)

| 屬性 | 路徑 | 型別 | 預設值 | 說明 |
|------|------|------|--------|------|
| show | `opt.panelScale.show` | `Boolean` | `true` | 是否顯示 |
| position | `opt.panelScale.position` | `String` | `'bottomright'` | 位置 |

- 使用 `l-control-scale`，且 `imperial=false`（僅公制）
- 自訂 CSS 樣式（深色底、白色文字、陰影）

---

## 5. 地圖資訊面板 (panelLabels)

| 屬性 | 路徑 | 型別 | 預設值 | 說明 |
|------|------|------|--------|------|
| show | `opt.panelLabels.show` | `Boolean` | `true` | 是否顯示 |
| position | `opt.panelLabels.position` | `String` | `'topright'` | 位置 |
| title | `opt.panelLabels.title` | `String` | `''` | 標題文字 |
| lng | `opt.panelLabels.lng` | `String` | `'Longitude'` | 經度標註文字 |
| lat | `opt.panelLabels.lat` | `String` | `'Latitude'` | 緯度標註文字 |
| zoom | `opt.panelLabels.zoom` | `String` | `'Zoom'` | 縮放標註文字 |
| useItems | `opt.panelLabels.useItems` | `Array` | `['lng','lat','zoom']` | 顯示項目 |
| width | `opt.panelLabels.width` | `Number` | `null` | 寬度 (px) |
| maxWidth | `opt.panelLabels.maxWidth` | `Number` | `null` | 最大寬度 (px) |
| height | `opt.panelLabels.height` | `Number` | `null` | 高度 (px) |
| maxHeight | `opt.panelLabels.maxHeight` | `Number` | `null` | 最大高度 (px) |

### 顯示內容

- 滑鼠移動時即時更新經緯度（`updateLatLng` 方法）
- 經緯度保留 7 位小數（使用 `dig` 函數）
- 縮放層級顯示當前 `zoom` 值
- 可透過 `title` 顯示標題（有時顯示分隔線）

## 6. 圖層顯隱面板 (panelItems)

| 屬性 | 路徑 | 型別 | 預設值 | 說明 |
|------|------|------|--------|------|
| show | `opt.panelItems.show` | `Boolean` | `true` | 是否顯示 |
| position | `opt.panelItems.position` | `String` | `'topleft'` | 位置 |
| width | `opt.panelItems.width` | `Number` | `null` | 寬度 (px) |
| maxWidth | `opt.panelItems.maxWidth` | `Number` | `null` | 最大寬度 (px) |
| height | `opt.panelItems.height` | `Number` | `null` | 高度 (px) |
| maxHeight | `opt.panelItems.maxHeight` | `Number` | `null` | 最大高度 (px) |
| stopWheel | `opt.panelItems.stopWheel` | `Boolean` | `false` | 是否攔截滾輪事件 |

- 使用 `Checkboxs` 子組件實現多選
- 僅在有 items 時顯示 (`items.length > 0`)

---

## 7. 圖例面板 (panelLegends)

| 屬性 | 路徑 | 型別 | 預設值 | 說明 |
|------|------|------|--------|------|
| show | `opt.panelLegends.show` | `Boolean` | `true` | 是否顯示 |
| position | `opt.panelLegends.position` | `String` | `'bottomright'` | 位置 |
| width | `opt.panelLegends.width` | `Number` | `null` | 寬度 (px) |
| maxWidth | `opt.panelLegends.maxWidth` | `Number` | `300` | 最大寬度 (px) |
| height | `opt.panelLegends.height` | `Number` | `null` | 高度 (px) |
| maxHeight | `opt.panelLegends.maxHeight` | `Number` | `null` | 最大高度 (px) |
| stopWheel | `opt.panelLegends.stopWheel` | `Boolean` | `false` | 是否攔截滾輪事件 |

### 圖例內容

- 僅顯示等值線圖層 (contourSets) 的圖例
- 需至少有一個可見的 contourSet (`countVisible(contourSets) > 0`)
- 每個 contourSet 的圖例包含：
  - `title`：標題
  - `legendMsg`：說明文字
  - `legend[]`：色階列表，每項含 `color`, `low`, `up`, `delimiter`, `textExt`, `arrow`

## 8. 共同面板樣式 (.clsPanel)

```css
.clsPanel {
    display: flex;
    padding: 3px;
    box-shadow: 0 0 15px rgba(0,0,0,0.2);
    border-radius: 5px;
    font-size: 0.8rem;
    font-family: 'Microsoft JhengHei', 'Avenir', 'Helvetica';
}
```

---

## 9. 滾輪事件控制

各面板的 `stopWheel` 屬性控制是否攔截滾輪事件：
- `true`：面板內捲動不會觸發地圖縮放
- `false`：滾輪事件穿透至地圖
- 透過 `e.stopPropagation()` 實現
