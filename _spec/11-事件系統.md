# 11 - 事件系統

## 1. 地圖原生事件

### 縮放更新 (updateZoom)

- 觸發時機：使用者透過滾輪/按鈕縮放地圖
- 行為：同步更新 `opt.zoom` 與內部 `zoom`

### 中心更新 (updateCenter)

- 觸發時機：使用者拖曳地圖
- 行為：同步更新 `opt.center`（使用 `isEqual` 防止重複觸發）

### 滑鼠座標更新 (updateLatLng)

- 觸發時機：滑鼠在地圖上移動
- 行為：更新 `showLoc.lat` 與 `showLoc.lng`（保留 7 位小數）

---

## 2. 各圖層點擊事件

所有圖層的點擊事件透過 `opt` 傳入的回呼函數觸發。

### 點圖層 (clickPoint)

```js
opt.pointSetsClick = function({ ev, point, kpoint, pointSet, kpointSet, pointSets }) { }
```

### 折線圖層 (clickPolyline)

```js
opt.polylineSetsClick = function({ ev, polylineSet, kpolylineSet, polylineSets }) { }
```

### 多邊形圖層 (clickPolygon)

```js
opt.polygonSetsClick = function({ ev, polygonSet, kpolygonSet, polygonSets }) { }
```

### GeoJSON 圖層 (clickGeojson)

```js
opt.geojsonSetsClick = function({ ev, lat, lng, latLngs, geojsonSet, kgeojsonSet, geojsonSets }) { }
```

### 等值線圖層 (clickContour)

```js
opt.contourSetsClick = function({ ev, lat, lng, latLngs, polygonSet, kpolygonSet, contourSet, kcontourSet, contourSets }) { }
```

---

## 3. 內部資料變更事件

### 圖層顯隱更新 (updateItems)

- 觸發時機：使用者在圖層面板切換 checkbox
- 行為：同步內部資料與外部 `opt` 的 visible 屬性

### 等值線重繪 (contourRefresh)

- 觸發時機：LwContour 子組件完成等值線計算
- 行為：生成圖例資料並存入對應 contourSet

---

## 4. DOM 事件

### 容器尺寸變更 (resize)

- 觸發時機：v-domresize 指令偵測到容器尺寸變化
- 行為：呼叫 `mapObject.invalidateSize()` 通知地圖重繪

### 面板滾輪事件

- `handleWheelPanelBaseMaps`
- `handleWheelPanelItems`
- `handleWheelPanelLegends`

各面板可獨立設定 `stopWheel` 來控制是否攔截滾輪。
