# 02 - 底圖管理

## 1. 概述

底圖管理面板提供使用者在多種底圖之間切換，支援 **WMTS** 與 **WMS** 兩種圖層類型。

---

## 2. 面板設定 (panelBaseMaps)

| 屬性 | 路徑 | 型別 | 預設值 | 說明 |
|------|------|------|--------|------|
| show | `opt.panelBaseMaps.show` | `Boolean` | `true` | 是否顯示底圖選擇面板 |
| position | `opt.panelBaseMaps.position` | `String` | `'topleft'` | 面板位置 |
| baseMaps | `opt.panelBaseMaps.baseMaps` | `Array` | `defBaseMaps` | 底圖清單 |
| width | `opt.panelBaseMaps.width` | `Number` | `null` | 面板寬度 (px) |
| maxWidth | `opt.panelBaseMaps.maxWidth` | `Number` | `null` | 面板最大寬度 (px) |
| height | `opt.panelBaseMaps.height` | `Number` | `null` | 面板高度 (px) |
| maxHeight | `opt.panelBaseMaps.maxHeight` | `Number` | `300` | 面板最大高度 (px) |
| stopWheel | `opt.panelBaseMaps.stopWheel` | `Boolean` | `false` | 捲軸面板內是否攔截滾輪事件 |

### 位置可選值

- `'topleft'`
- `'topright'`
- `'bottomleft'`
- `'bottomright'`

---

## 3. 底圖資料結構 (baseMap)

每個底圖物件需包含：

```js
{
    name: 'OpenStreetMap',     // 底圖名稱（顯示於面板）
    url: '//{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', // 圖磚 URL
    type: 'wmts',              // 圖層類型，預設 wmts，另可為 'wms'
    colorShade: 'light',       // 底圖色調 ('light' | 'dark' | '')
    opacity: 1,                // 透明度 (0~1)
    visible: true,             // 是否顯示 (Radio 單選)
    layers: '',                // WMS 專用：圖層名稱
}
```

---

## 4. 預設底圖 (defBaseMaps)

| 名稱 | 類型 | 色調 | 預設可見 |
|------|------|------|----------|
| Mapbox (衛星) | WMTS | dark | ✅ |
| OpenStreetMap | WMTS | light | ❌ |
| OpenTopoMap | WMTS | light | ❌ |
| GoogleStreets | WMTS | light | ❌ |
| GoogleSatellite | WMTS | dark | ❌ |
| GoogleHybrid | WMTS | dark | ❌ |

---

## 5. 底圖切換機制

1. 面板使用 `Radios` 子組件實現單選切換
2. 切換時更新 `baseMap.visible` 屬性
3. 底圖使用 `colorShade` 屬性影響其他 UI（如羅盤圖標明暗）

---

## 6. 圖層渲染規則

### WMTS 圖層

```html
<l-tile-layer
    :name="baseMap.name"
    :url="baseMap.url"
    :opacity="baseMap.opacity"
    :zIndex="1+kbaseMap"
    :visible="baseMap.visible"
    v-if="baseMap?.type!=='wms'"
/>
```

### WMS 圖層

```html
<l-w-m-s-tile-layer
    format="image/png"
    transparent
    :baseUrl="baseMap.url"
    :layers="baseMap.layers"
    :opacity="baseMap.opacity"
    :zIndex="1+kbaseMap"
    v-if="baseMap?.type==='wms' && baseMap.visible"
/>
```

---

## 7. 資料保護機制

- 使用 `cloneDeep` 隔離外部與內部資料
- 底圖切換後保留使用者選擇，不因外部 opt 重設而覆蓋
- 面板設定變更時透過 `isEqual` 比對 `panelBaseMapsTemp`，僅在有差異時更新

---

## 8. 色調計算 (computed: baseMapColorShade)

- 遍歷所有 baseMaps，找出當前可見的底圖
- 回傳其 `colorShade` 值
- 用途：決定羅盤圖標使用亮色或暗色版本
