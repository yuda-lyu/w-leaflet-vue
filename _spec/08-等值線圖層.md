# 08 - 等值線圖層 (contourSets)

## 1. 概述

等值線圖層使用三角網格技術計算等值線，並以色階圖（色對映）方式呈現。
渲染由 `LwContour` 子組件處理，主組件負責資料傳遞與圖例管理。

---

## 2. 資料結構

### 等值線集合 (contourSet)

| 屬性 | 型別 | 預設值 | 說明 |
|------|------|--------|------|
| `title` | `String` | `''` | 集合標題 |
| `msg` | `String` | `''` | 集合說明 |
| `order` | `Number` | `null` | 排序用數字 |
| `points` | `Array` | `[]` | 資料點陣列（含座標與數值） |
| `thresholds` | `Array` | `[]` | 等值線門檻值（空陣列則自動計算） |
| `gradient` | `Object` | 見程式碼 | 色階設定 (0~1 → 顏色對映) |
| `getColor` | `Function` | `null` | 自訂取色函數 |
| `lineColor` | `String` | `''` | 框線顏色（空則自動取色） |
| `lineColorHover` | `String` | 同 lineColor | 滑鼠移入框線顏色 |
| `lineWidth` | `Number` | `1` | 框線寬度 |
| `lineWidthHover` | `Number` | `3` | 滑鼠移入框線寬度 |
| `fillOpacity` | `Number` | `0.2` | 填充透明度 |
| `fillOpacityHover` | `Number` | `0.5` | 滑鼠移入填充透明度 |
| `changeStyleWhenHover` | `Boolean` | `true` | 是否啟用 hover 樣式 |
| `legendNumDig` | `Number` | `null` | 圖例數字小數位數 |
| `legendTextFormater` | `Function` | `null` | 圖例文字格式化函數 |
| `legendTextExtra` | `Function` | `null` | 圖例額外文字函數 |
| `polygonClipOuter` | `Array` | `[]` | 剔除外部多邊形 |
| `polygonsClipInner` | `Array` | `[]` | 剔除內部多邊形 |
| `polygonsContainInner` | `Array` | `[]` | 保留內部多邊形 |
| `visible` | `Boolean` | `true` | 是否顯示 |

---

## 3. 色階系統

### gradient 物件

```js
{
    0:    '#0000ff',  // 最小值 → 藍色
    0.25: '#00ffff',
    0.5:  '#00ff00',
    0.75: '#ffff00',
    1:    '#ff0000'   // 最大值 → 紅色
}
```

### getColor 自訂函數

若提供 `getColor` 函數，將取代 gradient 色階，接收數值回傳顏色。

---

## 4. 圖例生成 (contourRefresh)

當 LwContour 子組件計算完成後，透過 `@refresh` 事件回傳等值線資料。

### 圖例項目結構

```js
{
    low,         // 下界值
    up,          // 上界值
    delimiter,   // 分隔符號（預設 '-'）
    textExt,     // 額外文字
    color,       // 色塊顏色
    arrow: false,// 是否顯示箭頭
    index         // 索引
}
```

### legendTextFormater

```js
legendTextFormater({ low, up, legends, index })
// 回傳 { low, up, delimiter }
```

### legendTextExtra

```js
legendTextExtra({ k, n, polygonSet })
// 回傳額外文字字串
```

### legendNumDig

使用 `dig` 函數取小數位數。

---

## 5. 空間裁剪

| 屬性 | 深度 | 說明 |
|------|------|------|
| `polygonClipOuter` | 2 | 剔除多邊形之外的等值線 |
| `polygonsClipInner` | 3 | 剔除多邊形之內的等值線 |
| `polygonsContainInner` | 3 | 僅保留多邊形之內的等值線 |

---

## 6. 事件

| 屬性 | 路徑 | 說明 |
|------|------|------|
| `contourSetsClick` | `opt.contourSetsClick` | 全域等值線點擊回呼 |

回呼參數（透過 `parseContourData`）：

```js
{
    ev,            // 原始事件
    lat,           // 點擊位置緯度
    lng,           // 點擊位置經度
    latLngs,       // 多邊形座標
    polygonSet,    // 當前多邊形
    kpolygonSet,   // 多邊形索引
    contourSet,    // 當前等值線集合
    kcontourSet,   // 集合索引
    contourSets    // 所有集合
}
```

## 7. 插槽

| 插槽名稱 | 作用域 | 說明 |
|----------|--------|------|

---

## 8. 圖例顯示規則

- 圖例面板僅在有至少一個可見的 contourSet 時顯示
- 圖例色塊由下往上排列（reverse）
- 支援多個 contourSet 同時顯示圖例（水平排列）
