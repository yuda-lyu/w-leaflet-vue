# 01 - 地圖基本設定

## 1. Props 入口

組件僅接受一個 prop：

```js
props: {
    opt: {
        type: Object,
        default: () => ({})
    }
}
```

---

## 2. 中心點 (center)

| 屬性 | 說明 |
|------|------|
| 路徑 | `opt.center` |
| 型別 | `Array` |
| 格式 | `[緯度(lat), 經度(lng)]` (WGS84) |
| 預設值 | `[23.5, 121.1]`（臺灣中部） |
| 驗證 | 必須為長度 2 的陣列，且兩個元素均為數字 |

### 行為規則

1. 外部變更 `center` 時，組件內部透過 `isEqual` 比對，僅在有差異時更新
2. 更新時會呼叫 `mapObject.panTo(center)` 強制平移地圖（解決 vue-leaflet 近距離移動失效問題）
3. 內部使用者拖曳地圖時，會透過 `updateCenter` 同步至外部 `opt.center`，避免外部重設 center 時覆蓋使用者操作
4. 內部儲存使用 `cloneDeep`，與外部記憶體脫勾

---

## 3. 縮放層級 (zoom)

| 屬性 | 說明 |
|------|------|
| 路徑 | `opt.zoom` |
| 型別 | `Number`（正整數） |
| 範圍 | `1 ~ 18` |
| 預設值 | `7` |
| 驗證 | 使用 `ispint` 檢查是否為正整數 |

### 行為規則

1. 外部變更 `zoom` 時，透過 `isEqual` 比對避免重複更新
2. 內部使用者滾輪或按鈕縮放時，透過 `updateZoom` 同步至外部 `opt.zoom`
3. 範圍限制：`Math.min(Math.max(zoom, 1), 18)`

---

## 4. 面板背景色 (panelBackgroundColor)

| 屬性 | 說明 |
|------|------|
| 路徑 | `opt.panelBackgroundColor` |
| 型別 | `String` |
| 預設值 | `'rgba(255,255,255,0.95)'` |
| 用途 | 所有控制面板（底圖選擇、圖層管理、資訊顯示等）的統一背景色 |

---

## 5. 初始化流程

```
mounted()
  │
  ├── 設定 debounce 實例 (dbcChangePointSets, dbcChangePolylineSets, ...)
  ├── 使用 $watch 動態註冊各數據集監聽
  │     ├── imageSets, imageSetsPopup
  │     ├── pointSets, pointSetsClick
  │     ├── polylineSets, polylineSetsClick
  │     ├── polygonSets, polygonSetsClick
  │     ├── geojsonSets, geojsonSetsClick
  │     └── contourSets, contourSetsClick
  └── 各 watcher 設定 { immediate: true, deep: true }
```

---

## 6. 銷毀流程

```
beforeDestroy()
  │
  └── 遍歷並解除所有動態 $watch (vo.wats)
```

---

## 7. DOM 自適應

- 使用自訂指令 `v-domresize` 偵測容器尺寸變化
- 觸發 `resize` 方法呼叫 `mapObject.invalidateSize()` 通知 Leaflet 重新計算尺寸

---

## 8. data 內部狀態

| 狀態 | 型別 | 說明 |
|------|------|------|
| `center` | `Array` | 當前地圖中心 `[lat, lng]` |
| `zoom` | `Number` | 當前縮放層級 |
| `panelBackgroundColor` | `String` | 面板背景色 |
| `showLoc` | `Object` | 滑鼠座標顯示 `{lat, lng}` |
| `wats` | `Array` | 動態 watcher 清單，用於銷毀時移除 |
| `dbcChange*` | `Function` | 各數據集的 debounce 函數實例 |
| `eff*Temp` | `Array/Object` | 各數據集的前次快照（用於 isEqual 比對） |
| `panel*Temp` | `Object` | 各面板設定前次快照 |
