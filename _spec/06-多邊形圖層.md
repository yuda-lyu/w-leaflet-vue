# 06 - 多邊形圖層 (polygonSets)

## 1. 資料結構

### 多邊形集合 (polygonSet)

| 屬性 | 型別 | 預設值 | 說明 |
|------|------|--------|------|
| `title` | `String` | `''` | 集合標題 |
| `msg` | `String` | `''` | 集合說明 |
| `order` | `Number` | `null` | 排序用數字 |
| `lineColor` | `String` | `'rgba(0,150,255,1)'` | 框線顏色 |
| `lineColorHover` | `String` | 同 lineColor | 滑鼠移入框線顏色 |
| `lineWidth` | `Number` | `3` | 框線寬度 |
| `lineWidthHover` | `Number` | 同 lineWidth | 滑鼠移入框線寬度 |
| `fillColor` | `String` | `'rgba(0,150,255,0.25)'` | 填充顏色 |
| `fillColorHover` | `String` | 同 fillColor | 滑鼠移入填充顏色 |
| `latLngs` | `Array` | `[]` | 多邊形座標（支援 polygon/multiPolygon） |
| `visible` | `Boolean` | `true` | 是否顯示 |

---

## 2. 樣式物件

### 一般樣式

```js
{
    fillColor,
    fillOpacity: 1,     // 強制為 1（覆蓋 vue-leaflet 預設 0.2）
    color: lineColor,
    weight: lineWidth
}
```

### Hover 樣式

```js
{
    fillColor: fillColorHover,
    fillOpacity: 1,
    color: lineColorHover,
    weight: lineWidthHover
}
```

---

## 3. 渲染

使用 `l-polygon`：

```html
<l-polygon
    :ref="polygonSet.id"
    :latLngs="polygonSet.latLngs"
    v-bind="polygonSet.style"
    @click="clickPolygon"
/>
```

---

## 4. 事件

| 屬性 | 路徑 | 說明 |
|------|------|------|
| `polygonSetsClick` | `opt.polygonSetsClick` | 全域多邊形點擊回呼 |

回呼參數：

```js
{
    ev,             // 原始事件
    polygonSet,     // 當前多邊形集合
    kpolygonSet,    // 集合索引
    polygonSets     // 所有集合
}
```

---

## 5. 插槽

| 插槽名稱 | 作用域 | 說明 |
|----------|--------|------|
| `polygon-tooltip` | `{ polygonSet, polygonSets }` | 自訂 Tooltip |
| `polygon-popup` | `{ polygonSet, polygonSets }` | 自訂 Popup |

使用 `TooltipRegion` 與 `PopupRegion` 子組件。
