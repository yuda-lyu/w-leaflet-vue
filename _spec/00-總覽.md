# WLeafletVue 組件規格文件 - 總覽

## 1. 組件用途

`WLeafletVue` 是一個基於 **Vue 2** 與 **Leaflet** 的地圖封裝組件，透過單一 `opt` prop 物件控制所有地圖行為與資料顯示。
支援多種地圖圖層類型、控制面板、互動事件，適用於 GIS 空間資料視覺化場景。

## 4. 組件架構圖

```
WLeafletVue (主組件)
├── Props: { opt: Object }
├── 子組件:
│   ├── LMap, LTileLayer, LControl, LControlAttribution, LControlZoom, LControlScale
│   ├── LLayerGroup, LMarker, LCircleMarker, LPolyline, LPolygon
│   ├── LImageOverlay, LWMSTileLayer
│   ├── LwGeoJson (自訂 GeoJSON 組件)
│   ├── LwContour (自訂等值線組件)
│   ├── Radios (底圖單選)
│   ├── Checkboxs (圖層複選)
│   ├── PopupPoint, PopupRegion (Popup 封裝)
│   └── TooltipPoint, TooltipRegion (Tooltip 封裝)
├── 指令:
│   └── v-domresize (DOM 尺寸變更偵測)
└── 外部資源:
    ├── defBaseMaps.mjs (預設底圖設定)
    └── uiRes.mjs (UI 圖標資源, base64)
```

## 5. 資料流向

```
外部 opt (prop)
    │
    ▼
changeOpt() ──→ 解析通用設定 (center, zoom, panels)
    │
    ├── changePointSetsDebounce() ──→ changePointSets() ──→ pointSets
    ├── changePolylineSetsDebounce() ──→ changePolylineSets() ──→ polylineSets
    ├── changePolygonSetsDebounce() ──→ changePolygonSets() ──→ polygonSets
    ├── changeGeojsonSetsDebounce() ──→ changeGeojsonSets() ──→ geojsonSets
    ├── changeContourSetsDebounce() ──→ changeContourSets() ──→ contourSets
    └── changeImageSetsDebounce() ──→ changeImageSets() ──→ imageSets
                                        │
                                        ▼
                                  changeItemsDebounce()
                                        │
                                        ▼
                                  changeItems() ──→ items (圖層顯隱清單)
```

## 6. 反應式更新機制

- `opt` 使用 `watch` 搭配 `immediate: true` 與 `deep: true` 監聽
- 各數據集（pointSets, polylineSets 等）使用 `$watch` 動態註冊監聽
- 所有更新均透過 `debounce` 機制防抖
- 資料變更時先透過 `isEqual` 比對避免不必要的重渲染
- 透過 `cloneDeep` 與外部 opt 記憶體脫勾，避免無限觸發

---

## 7. 座標系統

- 組件內部使用 **WGS84 座標系**
- 座標格式為 `[緯度(lat), 經度(lng)]`（Leaflet 慣例）
- 注意：GeoJSON 標準為 `[經度(lng), 緯度(lat)]`，組件內有做轉換處理
